# to-do-list-app

A simple todo list application

## Setup

``` bash
# clone repository
git clone https://github.com/genki-sano/todo-list.git

# change directory
cd todo-list

# install dependencies
npm i
```

## Usage

```bash
# build for production with minification
npm run build

# build for production with minification if there is a change
npm run watch
```
